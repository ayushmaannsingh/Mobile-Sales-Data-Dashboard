# Mobile-Sales-Data-Dashboard
📊 Mobile Sales Analytics Dashboard – Power BI Project

🚀 Project Overview

This project is an interactive Mobile Sales Analytics Dashboard developed using Power BI to analyze sales performance, customer behavior, and business trends in the mobile sales industry.

The dashboard provides meaningful insights through dynamic visualizations, helping businesses make data-driven decisions efficiently.

🔍 Key Insights
Identified top-performing mobile brands and models
Analyzed monthly sales and quantity trends
Compared customer payment preferences
Tracked customer ratings and transaction patterns
Visualized city-wise sales distribution using maps
📊 Dashboard Features
Interactive slicers and filters
KPI cards for Total Sales, Quantity, Transactions & Average Sales
Monthly sales trend analysis
Brand-wise and model-wise performance tracking
Payment method analysis
Customer ratings visualization
Geographic sales analysis using map charts
🛠️ Tools & Technologies Used
Power BI
DAX
Power Query
Data Visualization
Data Cleaning
📈 KPIs Included
💰 Total Sales: 769M+
📦 Total Quantity Sold: 19K
💳 Total Transactions: 4K
📊 Average Sales: 40K
🎯 Objective

The goal of this project is to transform raw sales data into actionable insights and build a professional business intelligence dashboard for effective decision-making.
